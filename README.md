# Currency Converter Application

## Overview
A desktop currency converter application built using C# and Windows Forms in Visual Studio. The project allows users to enter a currency value, select a conversion option, and view the calculated result through a structured desktop interface.

## Tools Used
- C#
- Visual Studio
- Windows Forms

## Features
- Converts currency values from user input
- Displays calculated conversion results
- Uses a clear desktop form interface
- Includes calculation logic for currency conversion
- Tested for input behaviour and output accuracy

## Skills Demonstrated
- C# programming
- Windows Forms development
- User input handling
- Calculation logic
- Testing and debugging
- Interface design

## Project Outcome
Delivered a working desktop application that demonstrates programming, problem-solving, testing, and user interface development skills.
